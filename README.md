## Installation
Clone or dowload project. Open in Xcode 9+ and run on device or simulator.

## Requirements
* Xcode 9
* iOS 11

## Usage
- Open application on device or simulator.
- Type text in search field.
- Select content.
- Open iTunes by tapping button at the bottom of the screen.
